Brookfield Zoo Website Wireframe

Recommendations for site preformance

  1. Add Expires headers. Expires headers allow components that require http requests to be cached, meaning less http requests the following times the site is loaded.
  2. Compress components. Using gzip, compressing compontents will reduce their size in turn taking less time to load them.

Recommendations for site accessibility

  1. Add alt tags to images. Alt tags will describe the image to a user that cannot load or cannot see the image, and will help search engines catagorize your image.
  2. Add lang attribute to html tag. Identifying the lang attribute as "en" will allow screen readers to read content in the correct language. This will also allow for automatic translation of content
